# Library Management System 📚

This project is a console-based library management system developed in Java. It is designed around Object-Oriented Programming (OOP) principles to ensure high maintainability, scalable architecture, and streamlined software deployment in the future.

**⚠️ Note: This project is currently under active development and maintenance.**

## 🚀 Features (Current & Upcoming)

- [x] Add, update, and remove books from the inventory.
- [x] Track the real-time status of books (available/issued).
- [ ] User (Member) management module (In Progress).
- [ ] Book borrowing and returning operations (In Progress).
- [x] User-friendly, interactive console interface.

## 💻 Technologies

- Java (JDK 8 or higher)
- Object-Oriented Design for sustainable maintenance.
